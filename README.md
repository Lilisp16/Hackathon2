# Agenda Telefónica - Hackatón

## Descripción
Proyecto de agenda telefónica en Java para hackatón.  
Permite gestionar contactos (nombre + teléfono) con funcionalidades como añadir, buscar, eliminar y listar contactos.  
Se puede crear la agenda con tamaño máximo definido o usar el tamaño por defecto de 10 contactos.

## Funcionalidades
- Añadir contacto (evita duplicados)
- Comprobar si existe un contacto
- Listar todos los contactos
- Buscar contacto por nombre
- Eliminar contacto
- Comprobar si la agenda está llena
- Ver cuántos espacios libres quedan
